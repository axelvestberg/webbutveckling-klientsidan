Font Family

Det finns tre fontfamiljer:

  • Serif
  • Sans-serif
  • Monospace

Serif har små linjer i slutet på vissa bokstäver

Sans-serif har inte små linjer i slutet på vissa bokstäver

Alla bokstäver i Monospace är lika breda


Enheter

Olika enheter kan t.ex. användas för att skriva fonts i olika storleker. Man kan använda xx-small, x-small, small, medium, large, x-large, xx-large. Eller så kan man använda % eller em för att ändra storleken på text.

Övrigt

Med color kan man göra texten till olika färger

Med font kan man ändra alla fontens egenskaper samtidigt
