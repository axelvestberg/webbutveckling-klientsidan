CSS används för att definiera stilar för dina webbsidor, inklusive design, layout och variationer i displayen för olika enheter och skärmstorlekar. Med en extern CSS-fil kan du ändra utseendet på en hel webbplats genom att bara ändra en fil!

Arv

Arv innebär att vissa CSS-egenskaper förs vidare till underliggande element.

Cascade

Cascade i det här sammanhanget innebär att eftersom mer än en CSS-regel kan gälla för en HTML-fil, måste det finnas en metod för att bestämma vilken CSS-regel som gäller för vilken del HTML-koden. Regeln som används väljs genom att man går ner från de mer allmänna reglerna till mer specifika regler. Den mest specifika regeln är väljs.

Selektorer

För att visa vilken CSS-regel som gäller för vilken HTML-kod så används olika selektorer t.ex tagg-, attribut-, klass-, id- och barn-selektorer.

Enheter

I CSS används olika enheter för mått, det finns två typer av mått, absoluta och relativa. Exempel på absoluta enheter är in, cm, mm, pt, pc, px. Dessa är lika stora oavsett storlek på skärmen. Exempel på relativa enheter är em, ex %. Dessa är olika stora beroende på miljön de är i och/eller storleken på skärmen.

Hur ett CSS-dokument är uppbyggt

Koden i CSS-filen är strukturerad så att man först anger selektorn på den HTML-kod man vill styla. Innanför curly brackets anger man sen properties och values. Nedan kommer ett exempel på hur ett CSS-dokument kan se ut.


        #span {
            color: red;
        }
        img {
            max-width:250px;
            max-height:200px;
            width:auto;
            height:auto;
        }
